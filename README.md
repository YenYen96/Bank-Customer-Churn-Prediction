# Churn Prediction Model

Machine learning project for detecting bank customers churn using Python.

## Project Files

- `Bank Customer Churn Prediction.ipynb`: Main notebook including data cleaning, EDA, feature engineering, model training, and evaluation.

## Dataset

The dataset is not uploaded directly to this repository because the file size is too large.

Dataset link: [Google Drive Dataset](https://drive.google.com/file/d/1aylKu0gc6kmkeNzTMyewKPSStNbPHicb/view?usp=sharing)

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Evaluation Metrics

- Precision
- Recall
- F1-score
- PR-AUC
- Confusion Matrix
